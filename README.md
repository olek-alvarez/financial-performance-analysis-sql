# Financial Performance Analysis with SQL

## Overview
This project analyzes financial performance across countries using the AdventureWorks dataset.

The goal is to identify:
- Which markets generate the highest revenue
- Which countries are the most profitable
- Where marketing investment is most efficient

---

## Tools Used
- SQL
- PostgreSQL (or similar)
- Data analysis techniques

---

## Dataset
- sales_2017
- products
- product_categories
- territories
- marketing_campaigns

---

## Key Metrics
- Revenue
- Cost
- Gross Profit
- Margin (%)
- ROI (%)
